description

You need a personal portfolio website, one page modern simple design, with a little scrolling 3d animation maybe this theme can be your alternative to start it all to give a cool presentation to your prospective clients what services you offer, this template was made with a framework that very popular bootstrap, jQuery and most importantly this page will be 100% responsive on all your client devices.

installation

extract the template package then copy paste / upload all files and folders in the package to the root folder of the hosting provider that you registered before, it needs to be made sure to make it easier I suggest you edit and test the theme according to your needs on your local computer before you go upload to your hosting service provider's server

if you want to run this theme on your local computer you need some third-party applications that can run local servers like XAMPP

Color

if you want to change the basic color of this theme, you can simply open it and edit the color code in the color.css file


you can change the HEX color code unit

--baseone: #fff;

you can use this free tool hex2rgba to find transaparant rgba code from hex code

Customize 3d effect

if you want to customize 3d effect basic color, wave animation, speed rotation etc of this theme, you can simply open it and edit the color code in the config.js file


change in this line

/* shapre demo config */
const configuration = {
wireframe: false,
positionX: 8,
positionY: 0,
rotateX: 1.0,
speedwave: 0.1,
quality: 100,
rotatespeed: 0.001,
modular: false,
colorsgradient: {
gradientone: '#ff5a00',
gradienttwo: '#0068ad',
gradienttree: '#960037'
}
};

Php Contact form


if you want to use the php contact form correctly in this theme, maybe you can follow the instructions below

Open the mail.php file which is inside the theme's zip package with your code editor and try to edit the following line

$to = 'e-mail@example.com'; //Your e-mail address

change it with your email address that you registered with your website host service provider
